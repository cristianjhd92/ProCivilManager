# Contribución
## Rama y commits
- Rama principal: `main` (protegida).
- Rama develop: `develop` (protegida).
- Estrategia: trunk-based con ramas cortas `feature/*`.

## Pull Requests
- Requerido: 1 review + CI ok.
- Adjuntar: motivo, alcance, cómo probar, riesgos.

## Definition of Done
- Compila y pasa lint.
- Pruebas manuales básicas.
- Doc actualizada si aplica (docs/*).
